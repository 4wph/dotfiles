  - *Resize window*: =Super= + =Mouse Right Drag=
  - *Change workspace*: =Super= + =1=-=9=

  - *Invoke dmenu*: =Super= + =D=

* Emacs
** Enviroment
   - *Open file*: =C-x C-f=
   - *Save file*: =C-x C-s=
   - *Switch buffer*: =C-x= =b=
   - *Kill buffer*: =C-x= =k=
   - *Switch window*: =C-x= =o=
   - *Kill window*: =C-x= =0=
   - *Bookmarks*: =C-x r=
     - *List*: =l=
     - *Mark*: =m=
     - *jump*: =b=
   - *Search*: =C-s=
   - *Reverse search*: =C-r=
   - *Search and replace*: =M-%=
   - *Comment region*: =M-;=
   - *Close Emacs*: =C-x C-c=

** Editing
   - *Go to end of file*: =M->=
   - *Go to start of file*: =M-<=
   - *Go to end of line*: =C-a=
   - *Go to start of line*: =C-e=
   - *Kill line*: =C-k=
   - *Start mark*: =C-SPC=
   - *Copy*: =M-w=
   - *Cut*: =C-w=
   - *Paste*: =C-y=
